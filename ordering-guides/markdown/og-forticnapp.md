Introduction & Overview

Product/Solution Description

FortiCNAPP (Cloud-Native Application Protection Platform) is a comprehensive, unified, data-driven platform that empowers
organizations to secure everything from code to cloud. Using both agent and agentless technologies, FortiCNAPP continuously
collects, correlates, and analyzes a broad range of data to provide security teams with the context needed to identify and
remediate risks faster. It offers an automated, end-to-end security solution for multicloud and hybrid infrastructures, helping
businesses evaluate architectures against best practices, identify vulnerabilities early in the CI/CD pipeline, and ensure
compliance.

FortiCNAPP also provides Data Security Posture Management (DSPM) capabilities that help organizations discover sensitive
data across cloud storage services and understand how that data may be exposed through identities, vulnerabilities, or
misconfigurations. This enables security teams to prioritize and remediate the data exposures most likely to result in a breach.

Key Highlights

•  Continuous Compliance: Ensure continuous compliance against major industry standards such as SOC2, ISO 27001, HIPAA,

HITRUST, PCI, NIST, FISMA, PII, and CIS Benchmarks.

•  Advanced Threat Detection: Features anomaly detection to reduce alert noise by automatically building baseline models of

behavior and alerting only on critical, anomalous events.

•  Risk Reduction & Prioritization: Surfaces only the most critical risks, eliminating alert fatigue, and provides context-rich

visualizations for quick action to prevent financial losses and brand damage.

•  Automated Security: Eliminates the need to write and maintain rules for threat detection.

•  Deep Visibility: Gain visibility across infrastructure, workloads, identities, and sensitive data from code to cloud.

•  Data Security Visibility: Discover and classify sensitive data across cloud storage services and understand how sensitive data

may be exposed through identity access, vulnerabilities, and misconfigurations.

•  Improved Productivity: Helps to investigate alerts 4x faster.

FortiCNAPP is a cloud-based SaaS solution, available for various cloud environments including Amazon Web Services (AWS),

CODE SECURITY FEATURES

Software Composition Analysis (SCA)

Identifies and analyzes open-source components and third-party libraries used in your applications to find known vulnerabilities. It also helps

organizations maintain an up-to-date Software Bill of Materials (SBOM) and manage open-source License Compliance risks.

Static Application Security Testing

Scans application's source code to find security vulnerabilities before the code is deployed. It uses sophisticated analysis to provide highly accurate

(SAST)

results with a low rate of false positives.

Infrastructure as Code (IaC) Security

Integrates with organization development process to identify misconfigurations and security risks in your IaC templates (e.g., Terraform,

CloudFormation) to prevent vulnerabilities from being deployed into the cloud environment.

Secrets Scanning

License Compliance

Part of the platform's attack path analysis, which visualizes how an attacker could exploit risks. It helps to identify and highlight exposed secrets—

such as API keys or credentials—that an attacker could use to gain access to your environment.

Tied to Software Composition Analysis and helps users assess and manage the open-source license risks associated with the third-party

components used in their applications.

Software Bill of Materials (SBOM)

Automatically maintains a real-time SBOM for every application. The SBOM is a comprehensive list of all the software components, including open-

source libraries, that make up your applications, which is essential for understanding your software supply chain and associated risks.

Dynamic Application Security Testing

A method of application security testing that simulates real-world attacks on a running application to find vulnerabilities. It is a “black box” approach

(DAST)

because it doesn’t require knowledge of the application’s internal code or structure, instead testing from an external, user-like perspective.

Appendices & Additional Information

Frequently Asked Questions (FAQs)

Is FortiCNAPP an agent-based or agentless solution?

It employs a combined agentless and agent-based approach for data collection.

What compliance standards does FortiCNAPP support?

It supports SOC2, ISO 27001, HIPAA, HITRUST, PCI, NIST, FISMA, PII, and CIS Benchmarks.

Does FortiCNAPP integrate with other security tools?

Yes, it integrates with tools like Jira, Slack, Splunk, ServiceNow, Datadog, etc.

How does FortiCNAPP reduce alert fatigue?

It uses Polygraph anomaly detection to surface only the most critical, anomalous events.

How does FortiCNAPP integrate with the Security Fabric?

FortiCNAPP currently has integrations with FortiGate, FortiAnalyzer, FortiSIEM and FortiSOAR.

What is Data Security Posture Management (DSPM)?

DSPM capabilities in FortiCNAPP help organizations discover, classify, and prioritize risks to sensitive data stored in cloud
environments. By correlating data locations with identities, vulnerabilities, and attack paths, FortiCNAPP helps prioritize the most
critical data risks.
